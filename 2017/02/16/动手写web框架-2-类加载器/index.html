动手写web框架(2): 类加载器

发表于 2017-02-16   |   分类于 动手写web框架

自定义一个类加载器，用于加载基础包名下的所有类，比如使用了某注解的类，实现了某接口的类，或者继承了某父类的所有子类等。

自定义注解

一个Web框架需要用到一些自定义的注解。

自定义类加载器

ClassHelper通过一个Set来维护基础包下的所有类，并提供Bean类、Service、Controller或者相应的注解类的获取方式。

ClassUtil工具类，提供与类操作相关的方法，比如加载类、获取指定包名下的所有类。
